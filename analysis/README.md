# Analysis Workspace

This folder is a lightweight sandbox for SQL exploration, notebook analysis, and tracking data observations.

- `sql/` → reusable exploratory SQL queries.
- `notebooks/` → ad-hoc analysis and quick visualizations.
- `notes/` → written insights, findings, and follow-up questions.
- `outputs/` → saved artifacts such as CSV exports from queries.

This directory is for exploration only and is **not** production pipeline code.
